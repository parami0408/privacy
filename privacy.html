隐私政策

更新日期：2026年4月21日

本应用（以下简称「我们」或「本应用」）是一款纯粹的本地自我管理工具，我们深知隐私对您的重要性，并承诺保护您的个人信息。

核心承诺： 本应用默认不收集任何个人信息，所有数据仅存储在您的设备本地或您指定的私有服务器。专慧体（注：“专慧体”为本应用独创的自适应自学习能力，区别于市场上常见的“智能体”，强调个性化、本地化与用户自主掌控）功能完全由您自主选配，需您明确授权方可进行数据交流。

1. 我们收集的信息

为保障应用基础功能正常运行，我们会在您明确同意后收集以下必要信息：

  • 设备标识符（Android ID）：用于唯一标识您的设备，区分不同用户，以便为每个设备独立配置管控策略。该标识不可回溯到个人身份，且仅存储在本地。
  • 应用列表：用于展示已安装应用，供您选择需要管控的应用（如设置时间限制或白名单）。
  • 应用使用情况：通过系统 UsageStats API 获取应用使用时长，用于实现防沉迷管控（如超时锁屏）。
  • 存储空间：用于备份和恢复策略配置文件（仅限您主动操作时）。

上述信息仅用于提供应用的核心功能，不会上传至任何服务器。

2. 权限说明

本应用是一款强管控工具，需要完整配置以下所有必要权限及系统加固措施（包括壁纸服务、电池白名单、自启动管理等），方能保障管控策略的持续有效。若仅开启部分权限（例如仅开启无障碍服务而未进行其他系统加固），当应用退出后台或被系统清理时，相关服务将自动停止，这是Android系统的安全机制，并非应用功能异常。

重要声明：您同意本隐私政策，即表示您理解并接受：为确保应用稳定运行，预防因Android系统限制（如省电策略、内存回收）导致的管控失效，本应用会在必要时自动检测缺失的权限或加固状态，并向您发送引导提醒。此提醒旨在帮助您获得完整体验，非骚扰行为，您可随时点击“稍后”忽略。

  • 设备管理器权限：用于防止应用被随意卸载，确保管控策略持续生效。您每天有特定窗口期（通常为2分钟）可自行卸载。此权限是后台保活的核心，必须与无障碍服务、壁纸服务等加固措施同时开启，否则退出应用后无障碍服务将失效。
  • 无障碍服务权限：用于检测当前前台应用、输入框敏感词，实现智能管控和防沉迷功能（如自动跳转学习应用）。本服务不会读取短信、通话记录。该服务本身易被系统回收，必须配合设备管理器、前台服务、电池白名单、壁纸服务等加固措施才能常驻后台。
  • 应用使用情况权限：用于统计各应用使用时长，按您的策略限制使用。
  • 显示在其他应用上层（悬浮窗）权限：用于实现锁屏界面和弹窗提醒。
  • 存储权限：用于备份和恢复策略配置（仅您主动操作）。
  • 壁纸服务及其他系统加固权限：用于防止Android系统省电策略或内存清理导致应用被强制停止。壁纸服务提供额外的后台运行上下文，是保障服务连续性的必要补充。我们会在应用内单独引导您完成这些加固配置。

3. 第三方服务与专慧体

本应用不使用任何第三方分析工具、广告SDK或社交媒体服务，不会主动与任何第三方共享您的数据。您自主选配的专慧体服务（注：“专慧体”为本应用独创的自适应自学习能力，区别于市场上常见的“智能体”，强调个性化、本地化与用户自主掌控）由您自行授权，数据交流仅在您与您的私有服务器之间进行，本应用不参与数据处理。

4. 数据存储与安全

所有数据均存储在您的设备本地（或您指定的私有服务器），我们无法访问或恢复您的私有数据。请妥善保管您的设备，避免数据丢失。如需云备份，建议使用您自己的私有云存储。

5. 儿童隐私

本应用不直接面向儿童，不会主动收集儿童个人信息。若您是未成年人，请在监护人指导下使用。

6. 您的权利

您可以在系统设置中随时撤销已授予的权限，或直接卸载本应用。卸载后，所有本地存储的配置数据将被清除。

7. 政策变更

若隐私政策发生重大变更，我们将在应用内显著位置提示您。本页面会持续更新，请定期查看。
